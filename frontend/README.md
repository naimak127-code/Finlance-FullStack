Finlance - Frontend (React.js)
This is the client-side application for the Finlance dashboard. It is built with React and Tailwind CSS, focusing on a clean, intuitive interface to help users manage their financial data without cognitive overwhelm.

Key Frontend Features
Interactive Dashboard: Dynamic charts showing income vs. expense trends using Chart.js.

Account-Based Views: Filter and view transactions specifically for HBL, Jazz Cash, or other accounts.

Real-time Budget Tracking: Visual progress rings that update instantly as expenses are logged.

Secure Auth Flow: Fully integrated login, registration, and profile management views.

Responsive Design: Optimized for various screen sizes using Tailwind's utility-first framework

Tech Stack
Framework: React.js

Styling: Tailwind CSS

Icons: Lucide React / FontAwesome

Charts: Chart.js / Recharts

State Management: (e.g., React Context API or Redux)

Folder Structure
src/components: Reusable UI elements (Buttons, Cards, Inputs).

src/pages: Main views (Dashboard, Transactions, Settings).

src/assets: Images and custom styling configurations.

src/services: API calling logic to communicate with the FastAPI backend.

Development Setup
Install Dependencies:

Bash
npm install
Run the Project:

Bash
npm start
